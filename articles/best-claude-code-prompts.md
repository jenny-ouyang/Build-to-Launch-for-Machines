## Core Philosophy

**Your quality isn't prompt-based. It's context-based.**

The Voice Extraction prompt works because it analyzes 5 real articles and produces a portable artifact (Voice Card) that persists across every session. The Workflow Orchestrator prompt works because it collapses a 7-step multi-tool workflow into one command with quality gates.

Most prompt lists overwhelm. This one shows:
1. The workflow you're actually in (not the task category)
2. How much time it earns back (per week)
3. What makes it work (context artifacts, not clever phrasing)

## Writing Prompts (4 prompts, ~7.5 hrs/week earned)

1. **Voice Extraction (~1 hr/wk)** — Analyzes 5 articles, produces Voice Card (cadence, sentence structure, vocabulary patterns). Artifact persists across sessions. Jenny caught AI-induced drift in her own writing via this prompt.

2. **Voice-Checked Humanization Pass (~1 hr/wk)** — Takes AI draft + Voice Card, rewrites to match actual human patterns. Removes AI tells, matches sentence rhythm.

3. **Article Pipeline (>5 hrs/wk)** — 7-phase workflow with skip logic: research → outline → draft → fact-check → voice → SEO → publish. Runs in 20 min on re-runs vs 2-3 hours first run.

4. **Social Notes (~30 min/wk)** — Takes article, generates 3 note variants (tactical, worldview, question). Built-in engagement formula.

## Research Prompts (4 prompts, ~10 hrs/week earned)

5. **Idea Validation (>2 hrs/wk)** — Competitive analysis + market gap identification + red flag detection. Jenny validated her own product (16 sites, 7 usable data sources) in 70 minutes.

6. **Deep Research (~3 hrs/wk)** — Consolidates 100+ credible sources into one structured file in 5-20 minutes. Citation-backed, bias-checked.

7. **Competitor Strategy (~3 hrs/wk)** — Analyzes competitor content, positioning, gaps. Outputs differentiation opportunities.

8. **Fact Verification (~2 hrs/wk)** — Cross-checks claims against multiple sources. Returns confidence scores + citations.

## Coding Prompts (4 prompts, ~10.5 hrs/week earned)

9. **TDD Build (~3 hrs/wk)** — Test-Driven Development workflow. Feature built and tested in <10 min when broken into modules.

10. **Debugging (~4 hrs/wk)** — Systematic debugging protocol: reproduce → isolate → test hypothesis → verify fix.

11. **Refactoring (~2 hrs/wk)** — Code smell detection → improvement suggestions → refactor execution.

12. **Clean Commits (~1.5 hrs/wk)** — Git commit message generation with approval gate. "Saved me dozens of times" from shipping broken commits.

## Systems Prompts (3 prompts, ~5.5 hrs/week earned)

13. **Structural Audit (~1 hr/wk)** — Analyzes codebase/content architecture. Finds redundancies, missing patterns, organization gaps.

14. **SEO Analysis (~1 hr/wk)** — Title optimization, meta descriptions, internal linking opportunities. GSC data integration.

15. **Workflow Orchestrator (~2.5 hrs/wk)** — Multi-tool workflow automation with quality gates. 7-step publishing pipeline example: skip completed phases, halt on failed checks.

## The Prompt Progression

**Level 1: Copy-paste** — Use prompts as-is, modify for your context.

**Level 2: Slash commands** — Convert repeated prompts to `/shortcuts` in Claude Code.

**Level 3: MCP-connected prompts** — Connect prompts to external tools (Notion, Supabase, Stripe). Context auto-loads.
